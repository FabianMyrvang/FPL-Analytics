# Historical Seed Data

This folder contains the historical seed datasets (2020–21 through 2024–25 seasons) generated by `scripts/01_fpl_merge.py` from the [vaastav/Fantasy-Premier-League](https://github.com/vaastav/Fantasy-Premier-League) archive.

These files bootstrap the pipeline — they provide the persistent player/team ID mappings and baseline history that `scripts/02_fpl_api.py` upserts onto during the live season.

| File | Description |
|---|---|
| `fpl20_24.csv` | Per-player per-gameweek FPL data, 2020–21 through 2024–25 |
| `players.csv` | Player ID ↔ full name mapping (persistent cross-season) |
| `teams.csv` | Team ID ↔ team name mapping (persistent cross-season) |
| `positions.csv` | Position ID ↔ position code (GK/DEF/MID/FWD) |
| `fixtures.csv` | Historical fixtures with persistent match IDs |
| `fixtures_stats.csv` | Per-match aggregated team stats |

These are **static** — they don't change unless you regenerate them by running `scripts/01_fpl_merge.py` with a fresh clone of the vaastav archive. For normal pipeline runs, they're used as-is.

See `docs/fpl_merge_pipeline.md` for full documentation.
